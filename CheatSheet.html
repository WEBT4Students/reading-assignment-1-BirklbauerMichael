World Wide Web

  1. What are the most essential technologies that made the World Wide Web such a success?
    2. The most essential technologies are HTML, URL and HTTP.

  2. How are resources (html pages, images, etc) addressed in the web?
    3. Resources are addressed with URI.

  3. What are the general components of a url?
    4. The URL consists of a hierarchical sequence of five components: sheme, authority, path, query and fragment.

Web Server

  1. What is a web server?
    2. A web server is computer software and underlying hardware that accepts requests via HTTP.

  2. What does a web server?
    3. A user agent, commonly a web browser, initiates communication by making a request for a web page or other resource using HTTP, and the server responds with the content of that resource.

  3. How can content be stored on a web server?
    4. The web space is a hard disk, on which web content is stored and this must be connected to the web server.

  4. How is content in webpages described?
    5. Page content refers to all the information contained in a website. Page content can be displayed as text, links, images, audio, animation or videos among other things.

Web Client

  1. What is a web client?
    2. A web browser is application software for accessing the World Wide Web or a local website.

  2. What does a web client?
    3. When a user requests a web page from a particular website, the web client retrieves its files from a web server and then graphically renders the page on the user's screen.

  3. Web Client and Web Browser is the same?
    4. The web browser acts as an interface between the server and the client and displays a web document to the client.

Hello World

  1. What is the html head element?
    2. The head element is a container for metadata. Metadata is data about the HTML document. Metadata typically define the document title, character set, styles, scripts, and other meta information.

  2. What does !DOCTYPE html mean?
    3. All HTML documents must start with a !DOCTYPE declaration. It is an information to the browser about what document type to expect.

  3. What are the two top most elements within the html element?
    4. The two top most elements within the html element are head and body.

  4. How to emphasize a text and how is it displayed in the browser?
    5. The em tag is used to define emphasized text. The content in the browser is typically displayed in italic.

  5. How to mark up important text and how is it displayed in the browser?
    6. The strong tag is used to define text with strong importance. The content in the browser is typically displayed in bold.
